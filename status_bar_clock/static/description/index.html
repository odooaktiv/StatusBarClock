Status Bar Clock

The module will display the date and time directly from any module. The user can edit the default date and time as per their location and their format.

Prerequisites before Installation

No prerequisites needed.

Configurations

No configurations.

Once you install the module, default date and time will be displayed according to a system on the screen.

You can edit it from: - Settings>Users>Company> Clock format.

As soon as you update the date and time from the settings you can see the date and time displayed in the left of the status bar. Suppose if a user has multiple companies with different location, then he/she can provide different date and time for each company. If you switch the company the date and time will be updated accordingly.
