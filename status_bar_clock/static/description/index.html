Status Bar Clock

This module helps user to check date and time directly from any module.

  • After installing the module there will be default date time format but if you want to change format you can do it from settings > users > companies > clock format.It will be shown according to your format.
  • The clock will be displayed on left side in status bar.Date and time will be displayed according to system's date and time.If You will change the company, time will automatically change according to it.
